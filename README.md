# Fake News Detector

A simple web-based fake news detector using keyword analysis (no backend required).

## Live Demo

Visit: https://samyamoy1.github.io/fake-news-detector-demo/

## How It Works

This uses a keyword-based approach to detect potential fake news by analyzing:
- **Sensational words** (breaking, shocking, secret, miracle, etc.)
- **Formal/neutral words** (announces, research, official, report, etc.)

The detector counts matches and determines if the text is more likely FAKE or REAL.

## Features

- Simple keyword-based analysis
- No backend required - runs entirely in browser
- Works on GitHub Pages
- Shows confidence percentage
